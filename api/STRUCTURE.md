FOLDERS
app folder - contains the pages of the page
    page.tsx - content of the page
    layout.tsx - layout of the page
components - contains the reusable components used in the page
drizzle - generated folder for the database
hooks - reusable functions used in the app. mostly for fetching and creating
lib
    api - used for the fetching from the app
    db - database related functions
public - files that can be accessed publicly
scripts - used for the scripts run in the server


FILES 
middleware.ts - config for middleware like auth
